CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. It allows you to control the layout, colors, fonts, and more.

With CSS, you can make your website visually appealing and responsive.

Here are some basic CSS properties:

  • color: Sets the text color.
  • background-color: Sets the background color.
  • font-size: Sets the size of the font.
  • margin: Sets the space outside an element.
  • padding: Sets the space inside an element.
  • border: Sets the border around an element.
  • display: Controls the display behavior of an element.
  • flex: Enables flexible layouts.
  • grid: Enables grid layouts.
  • position: Controls the positioning of an element.
  • float: Allows elements to float next to each other.
  • text-align: Sets the alignment of text.
  • width: Sets the width of an element.
  • height: Sets the height of an element.
  • overflow: Controls what happens when content overflows an element.
  • opacity: Sets the transparency of an element.
  • transition: Adds transition effects to elements.
  • transform: Applies 2D or 3D transformations to elements.
  • animation: Adds animations to elements.
  • box-shadow: Adds shadow effects to elements.
  • text-shadow: Adds shadow effects to text.
