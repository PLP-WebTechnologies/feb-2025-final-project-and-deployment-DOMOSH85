Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the structure of a webpage using elements like headings, paragraphs, links, and more.

HTML is easy to learn and is the foundation of web development.

Basic HTML Structure

An HTML document starts with a declaration, followed by the element. Inside the element, you have the and sections.

The section contains meta-information about the document, such as the title and links to stylesheets. The section contains the content that is displayed on the webpage.

HTML Elements

HTML elements are the building blocks of a webpage. They are defined by tags, which are enclosed in angle brackets. For example, a paragraph is defined using the

tag.

Here are some common HTML elements:

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (link)
  • <img>: Image
  • <div>: Division (block-level element)
  • <span>: Span (inline element)
  • <ul>: Unordered list
  • <ol>: Ordered list
  • <li>: List item
  • <table>: Table
  • <form>: Form
  • <input>: Input field
  • <button>: Button
  • <header>: Header section
  • <footer>: Footer section
  • <section>: Section of content
